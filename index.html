Music Composition with Quantum Computers

QINTERN 2021

Research Project led by Srinjoy Ganguly

It is a familiar concept that musicians and sound engineers rely a lot in various mathematical and scientific research to forage novel techniques which can help them to compose and generate novel music and sound patterns. This project is one of the first unique kind of project which is based on music composition using quantum circuits. Since this is a very new field, therefore, the aim of this project will be to utilize all the resources to construct invaluable educational tutorials and materials based on quantum computing for music composition.

We are working on creating a website which explains the entire concept of Music Composition with Quantum Computers in Laymanâ€™s terms, right from the classical composition of music to tutorials on different software packages which allow you to play around with circuits to compose music.

This website has different chapters which can be accessed by clicking the menu. They are arranged in such a way so as to give a complete overview of how exactly music can be composed with Quantum Computers.

We have been working on this website as a part of QWorld's QIntern 2021 program. This project is done under the guidance of Srinjoy Ganguly.
